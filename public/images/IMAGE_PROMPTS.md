## Style Guide for All Images

- **Primary background**: #0a0a0f (near black)
- **Surface colors**: #111118, #18181f
- **Accent color**: #f97316 (warm orange)
- **Secondary accent**: #eab308 (gold, for crafting)
- **Text-safe zones**: Leave space for text overlays on wide/banner images
- **Format**: PNG with transparency where noted, JPEG for backgrounds
- **Consistency**: All images should feel like they belong to the same dark fantasy UI toolkit
